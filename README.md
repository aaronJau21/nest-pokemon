# Pokemon

1. Clonar el repositorio

```
git clone "url"
```

2. Ejecutar

```
yarn install
```

3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```

## STACKS

* NEST JS
* Mongo DB
* Docker
